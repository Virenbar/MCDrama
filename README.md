# MC Drama

Ruby for generating MC drama

## Notes to myself

Instruction for hosting on NetAngels

1. Turn site off
2. Change ruby to 2.7
3. SSH to container and run commands

    ```sh
    git clone https://github.com/Virenbar/MCDrama.git app
    cd ./app
    gem install bundler:1.17.3
    bundle install
    ```

4. Turn site on
